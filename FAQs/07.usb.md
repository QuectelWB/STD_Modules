correlate it with lsusb. A simple shell script which will print only RootHubs available on your machine.

```
#!/bin/sh
TOTAL_BUS_NUM=$1
for (( i=1; i <= $TOTAL_BUS_NUM; i++ ))
do
    lsusb -s $i:1
done
```
